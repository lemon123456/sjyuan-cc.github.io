

## Ruby(001) Get Started
@(Ruby)[Ruby|Get Started]

### Install Ruby with [`rvm`](https://rvm.io/)
* `$ brew install ruby`
* `$ rvm install 2.2.3`
    * $ rvm list
    * $ rvm use 2.1.3
    * $ rvm --help

>可以使用rvm来管理ruby的版本，更多关于`RVM`内容，[请点我](https://rvm.io/)

---

### Use Ruby	，
* `$ ruby -v`
* `$ irb`

>`ruby -v`可以查看当前Ruby版本号  
>`irb`打开交互式命令行窗口

---

### View Ruby Doc
#### Run server
* `$ ri --server`

####
